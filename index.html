Getting Started

Setup is easy and designed to be highly modular. Just follow these steps in order:

  1. Optional Include the Internet Explorer Shiv (read why here)
  2. Optional Include jQuery if using jQuery Plugins
  3. Optional Include any jQuery(UI) plugins you wish to use (plugins are not bundled with AngularUI)
  4. Include AngularJS
  5. Include AngularUI in one of the following ways:
    • Include the entire library (located in /build directory)
    • Include the /common/module.js and any directives or filters you wish to use
    • Create a custom build and include it
  6. Add the 'ui' module as a dependency: 
    angular.module('myModule', ['ui'])
  7. Optional Setup your Global Defaults for plugins and directives

Global Defaults

You can specify default options for most AngularUI directives. This way you don't have to pass them every single time you call the directive. Passed options will be merged onto defaults, overriding any pre-existing keys.

Simply declare the global config on your module: module.value('ui.config', {}). Each directive should be namespaced by the directive name (without the ui- prefix).

Here's an example to make ui-select2 clearable by default:

var myModule = angular.module('myModule', ['ui']);
myModule.value('ui.config', {
   select2: {
      allowClear: true
   }
});
